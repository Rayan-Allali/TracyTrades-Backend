
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Prerequisites

This project leverages [Prisma](https://www.prisma.io/) for database access and [GraphQL](https://graphql.org/) for efficient API development.
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
